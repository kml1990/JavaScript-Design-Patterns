# JavaScript-Design-Patterns

## Creational 

Patterns that deal with new instances of an object
- Constructor 
- Module
- Factory 
- Singleton 

## Structural 

Deal with a make up of actual objects themselfs
- Decorator
- Facade
- Flyweigh

## Behavioral

Deal with how object deal with each other and how they operate
- Command 
- Mediator
- Observer
